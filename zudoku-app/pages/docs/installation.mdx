## Prerequisites
Before using the Housing Rental System SDK, ensure you have the following installed:

- **.NET 6+** (for C# SDK) - [Download .NET](https://dotnet.microsoft.com/download)
- **Node.js 16+** (for JavaScript SDK) - [Download Node.js](https://nodejs.org/)

## Clone repo [API](https://github.com/mamay12/POPIM) and run server

## Install and run openapi-generator

```
npm install @openapitools/openapi-generator-cli -g
```
Then simply generate SDK using command
```
openapi-generator-cli generate -i http://localhost:5137/swagger/v1/swagger.json -g csharp -o ./generated-client
```
